# Kamala Harris for Governor of Lincoln

A polished, responsive campaign website built with React, TypeScript, Vinext/Vite, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open the local address shown in the terminal. Create a production build with `npm run build`.

## Included

- Responsive campaign masthead and mobile navigation
- Homepage with candidate message, record, platform, news, events, endorsements, signup, and action rail
- Full About, Record, Issues, News, Endorsements, Events, Media, Get Involved, Donate, and Contact experiences
- Interactive issue tabs, record accordions, event and endorsement filtering, donation selection, form confirmations, and news article views

This project is a static front-end demonstration. Forms and donation controls do not transmit or retain information.
